Welcome!

If you are here, then you probably already know what skribbl.io is and how to play it. This is Yangfa's replica of this online drawing game. I built everything but the images, sounds, and game design from scratch. If you are more curious about the technical stuff, this was made using Angular and the game itself is currently backed by a remote pseudo-server written in Java. I have no idea how the original developer made the game but the end result is pretty similar. Enjoy!

Please remember to support the original game and its creator!
